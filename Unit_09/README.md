This directory contains all fils associated with Unit 09 assignments and documents. This assignment covers how to scrape web pages with R packages to generate data frames from web based sources. The rvest package was used to capture information from IMDB's website associated with Actors from the Harry Potter movies to answer Question 1. Similarly, shooting stats for the San Antonio Spurs was captured from ESPN's website for Question 2.
